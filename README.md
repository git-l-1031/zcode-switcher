# ZCode Switcher

简体中文 | [English](README.en.md)

ZCode 账号管理与切换桌面工具，支持额度显示、无感切换和应用内更新。

## 主要功能

- 本地保存并管理多个 ZCode 账号。
- 支持无感切换账号，支持低额度自动切换。
- 支持无感切换下的胶囊悬浮窗，显示 GLM-5.2 账号池统计。
- 显示和刷新账号额度信息。
- 支持中文、英文、俄文界面。
- 支持在软件内检测更新、下载并安装。
