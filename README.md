# ZCode Switcher

简体中文 | [English](README.en.md)

ZCode 账号切换与管理工具。无需重启！无需重启！无需重启即可切换账号，真正实现无感切换！支持自动切换！

## 主要功能

- 本地保存、json导出导入、管理多个ZCode 账号。
- 无需重启，后台无感切换账号。
- 支持低额度自动切换，自定义切换阈值。
- 新增无感切换下的胶囊悬浮窗，支持自定义大小，显示 GLM-5.2 账号池额度使用统计。
- 显示和刷新账号额度信息。
- 支持中文、英文、俄文界面以及深浅风格切换。
- 配置目录可见，数据完全本地存储，安全可靠。
- 支持在软件内检测更新、下载并安装。
