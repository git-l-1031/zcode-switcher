# 更新日志

## 1.1.5

- 增加了 OAuth 方式添加账号。

## 1.1.4

- 优化刷新逻辑。

## 1.1.3

- 优化无感切换刷新方式及卡顿问题，开启后自动配置快捷方式参数。
- 优化悬浮窗排版，账号池信息显示更紧凑。
- 优化自动重启逻辑，尽量保留无感切换所需参数。
- 改进配置写入和切号流程。

## 1.1.2

- 增加功能栏隐藏账号信息按钮，可隐藏账号信息。
- 移除功能栏当前登录提示，让顶部操作区更简洁。
- 降低偶发限流导致刷新失败的概率。
- 优化卡片布局，整体观感更整齐。

## 1.1.1

- 修复检测更新弹窗在透明窗口下的闪烁问题。
- 下载更新时，在弹窗内显示进度条并自动安装重启。
- 启动时静默检测新版本，并在设置图标旁显示更新提示。
- 优化批量刷新额度节奏，降低后台请求压力。

## 1.1.0

- 优化悬浮窗拖动和透明圆角外观，避免文字选中影响拖动。
- 支持在悬浮窗内调节窗口大小。
- 增加批量删除账号功能，支持多选删除本地账号档案。
- 增加一键删除订阅过期或额度获取错误账号。
- 检测到新版本时，下载确认弹窗会显示该版本更新内容。

## 1.0.3

- 添加悬浮窗模式。

## 1.0.2

- 添加检测更新功能。

## 1.0.1

- 添加无感切换相关功能。

## 1.0.0
