# 使用说明

## 悬浮窗模式

先在设置中开启“无感切换”，再开启“悬浮窗模式”。

开启后，ZCode Switcher 会切换成椭圆胶囊窗，显示 GLM-5.2 账号池统计：

- 已用账号数量：GLM-5.2 剩余额度低于当前设置阈值的账号数。
- 剩余账号数量：总账号数减去已用账号数。
- 已用累计额度：所有账号 GLM-5.2 已用额度累计。
- 总额度：所有账号 GLM-5.2 总额度减去“账号数 × 当前阈值”，再加上已用账号低于阈值的部分。

点击胶囊右侧关闭按钮，窗口恢复普通模式。

## 隐藏账号信息

点击主界面功能栏里的眼睛按钮，可以隐藏账号列表中的邮箱和手机号区域。

再次点击该按钮，会恢复显示账号信息。
