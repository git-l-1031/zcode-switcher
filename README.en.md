# ZCode Switcher

[简体中文](README.md) | English

A desktop tool for managing and switching ZCode accounts, with quota display, seamless account switching, and in-app updates.

## Features

- Save and manage multiple ZCode accounts locally.
- Seamlessly switch accounts and automatically switch when quota is low.
- Display and refresh account quota information.
- Support Chinese, English, and Russian interfaces.
- Check for updates, download, and install updates inside the app.
